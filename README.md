# Shear Chaos

Survive as a sheep caught in the crosshairs of a fierce battle!

A continuation of Godot's "first 2D game" tutorial.

## File structure

- `root`
 	- `assets`
  		- Assets used in game
 	- `src`
  		- Main game logic/code
  		- Godot scenes and script files
